## Contributing

Interested in contributing? Here's how to get started:

1. **Read** [CONTRIBUTING.md](CONTRIBUTING.md)
2. **Pick** an issue labeled "good first issue" or "help wanted"
3. **Fork** the repository
4. **Create** a feature branch
5. **Make** your changes
6. **Test** thoroughly
7. **Submit** a pull request

For questions, open a GitHub Discussion or contact the maintainers.

## Code of Conduct

All contributors are expected to follow our code of conduct as outlined in [CONTRIBUTING.md](CONTRIBUTING.md). We strive to maintain a welcoming, inclusive, and professional community.

Key principles:
- **Be respectful** and professional
- **Prioritize patient safety** in medical AI
- **Follow HIPAA guidelines** for data privacy
- **Maintain scientific rigor**
- **Collaborate openly**
